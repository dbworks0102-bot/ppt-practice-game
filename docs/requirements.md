## 1. プロジェクト概要

### 1.1 プロダクト名
**PPT道場 〜パワーポイント操作ゲーム〜**

### 1.2 目的
PowerPoint のショートカットキーを楽しみながら学べるブラウザゲームを WEB に公開する。  
寿司打ゲームのように、スピード・正確性に応じてポイントが加算されるゲーム形式で学習効果を高める。

### 1.3 ターゲットユーザー
| ユーザー層 | 特徴 |
|---|---|
| 社会人・学生 | PowerPoint を業務・学習で使うが、ショートカットに不慣れ |
| 自己学習者 | 繰り返しプレイして弱点を克服したい |
| 競争好き | ランキングに名を残したい |

## 2. 機能要求

### 2.1 コア機能（実装済み）

| ID | 機能名 | 内容 |
|---|---|---|
| F-01 | ゲームプレイ | ショートカットキーを4択から選択して回答 |
| F-02 | スコアリング | 回答速度に応じたポイント加算（×1.0〜×3.0） |
| F-03 | コンボシステム | 連続正解でポイント倍率アップ |
| F-04 | 難易度選択 | かんたん / ふつう / むずかしい の3段階 |
| F-05 | 時間ペナルティ | 不正解・時間切れで残り時間 −2秒 |
| F-06 | ゲームタイマー | 30秒制（難易度共通） |
| F-07 | ランキング | 難易度別・最大20件をlocalStorageに保存 |
| F-08 | プレイヤー名登録 | localStorageに保存・ランキングと紐付け |
| F-09 | やり直し・終了 | ゲーム中にモーダル経由でリスタート or 終了 |
| F-10 | マイページ | カテゴリー別正答率・苦手問題TOP10を表示 |
| F-11 | ショートカット問題 | 126問・11カテゴリ・難易度Lv1〜3 |

## 3. 非機能要求

| 区分 | 要求 | 目標値 |
|---|---|---|
| パフォーマンス | 初期ロード時間 | 3秒以内（3G回線） |
| 互換性 | 対応ブラウザ | Chrome / Firefox / Safari / Edge 最新版 |
| 互換性 | 対応デバイス | PC（主）・タブレット（補） |
| セキュリティ | データ保存 | localStorageのみ（個人情報なし） |
| アクセシビリティ | キーボード操作 | 1〜4キーで回答可能 |
| 保守性 | テストカバレッジ | 主要フロー100%（Playwright） |
| 可用性 | ホスティング | 静的ファイル配信（Vercel / GitHub Pages） |

---

## 4. 制約条件

- バックエンドサーバーを持たない（静的サイト）
- スコア永続化はlocalStorage（オンラインランキングは別フェーズ）
- フレームワークなし or 最軽量（バンドルサイズ最小化）

---

## 5. 用語定義

| 用語 | 定義 |
|---|---|
| SS中 | スライドショー中の操作を指す |
| コンボ | 連続正解数 |
| 速度ティア | 回答速度に応じた倍率区分（⚡超速／🔥速い／✨Good／👍正解／✅正解） |
| sessionStats | 1ゲーム内の問題別正解・不正解カウント |
| playerStats | プレイヤー累計の統計データ（localStorageに保存） |
